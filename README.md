# Парсинг книг с Books to Scrape

## Цель проекта
Разработать систему для автоматического сбора данных о книгах с сайта [Books to Scrape](http://books.toscrape.com). Проект включает функции парсинга страниц, извлечения информации о книгах, многопоточной обработки и автоматического запуска по расписанию.

## Инструкции по запуску
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите основной скрипт для парсинга всех книг: `python scraper.py`
3. Для запуска тестов: `pytest tests/test_scraper.py`

## Используемые библиотеки
- **requests** - выполнение HTTP-запросов
- **beautifulsoup4** - парсинг HTML-страниц
- **schedule** - автоматизация задач по расписанию
- **threading** - многопоточная обработка
- **pytest** - написание и запуск тестов
- **tqdm** - отображение прогресса выполнения
- **re** - работа с регулярными выражениями
- **json** - сохранение данных в формате JSON
- **time** - работа с временными интервалами
